# DutchyTPA
Minecraft 1.16, 1.17

This is a very, very simple TPA plugin.

## Commands
- ``/tpa <playername>``: Teleport to a player
- ``/tpaccept``, ``/tpyes``: Accept a TPA request
- ``/tpdeny``, ``/tpno``: Deny a TPA request

Told you, it's simple :)
